Watercolour basics

There are many simple basics that lay a foundation for good watercolour paintings.

Being able to paint a solid and smooth layer of watercolour is an effesential basic. It lets you create a solid base of colour and allows you to build on more colours.

By knowing how to blend watercolours allows for more exploration in colours and various inbetween shades. The best way to blend watercolour is to put colours next to eachother when it's still wet.

By using a wet brush on a dry patch of watercolour, it is possible to remove the layer to create fine details or help remove mistakes.
